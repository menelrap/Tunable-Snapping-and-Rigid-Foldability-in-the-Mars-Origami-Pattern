# Baseline Experiment Data (N = 31)

This folder contains the raw data files corresponding to the baseline experimental study of the 3 × 3 Mars origami tessellation.

## Description

A total of **31 independent experimental runs** were performed, all fabricated under identical laser-cutting and crease-scoring conditions. Each `.dat` file corresponds to one complete folding-unfolding cycle recorded using the force-displacement measurement system described in the paper.

The purpose of this baseline dataset is to characterize the intrinsic snapping behavior of the Mars pattern in the absence of any additional modifications (e.g., no added diagonal creases).

## Experimental Context

As described in the study, the Mars tessellation is **not rigidly foldable**, leading to geometric incompatibility that is resolved through facet bending. This produces a pronounced **snap-through instability** in the force-displacement response.

Across the 31 runs, the baseline behavior is highly reproducible:

- Mean force drop: **92.6 ± 5.5%**
- Mean snap position: **x ≈ -0.039 ± 0.002 m**
