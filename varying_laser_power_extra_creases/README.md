# Varying Laser Power on Added Diagonal Creases

This folder contains the experimental data for Mars pattern specimens in which additional diagonal creases were introduced on the square faces and engraved at different laser power settings.

## Description

Following the baseline experiments, supplementary diagonal creases were added to the square faces of the 3 × 3 Mars pattern, guided by strain-field visualizations obtained from the origami simulation. The goal was to test how the stiffness of these added creases influences the magnitude of the snap-through instability.

The additional diagonal creases were engraved using different Glowforge laser power settings:

- 2%
- 4%
- 6%
- 8%
- 10%
- 12%

Each dataset in this folder corresponds to a specimen tested under one of these laser-power conditions.  
**The specific laser power used in each experiment is indicated directly in the filename.**
